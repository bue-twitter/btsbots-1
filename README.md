# btsbot
bots for BitShares blockchain

## How to use
- Install all dependencies
```
$ npm install
```
- Compile source
```
$ npm run build
```
- Set workers in `config.js`
- Start workers
```
$ npm start
```
## Development mode

```
$ npm run dev
```
